# Nikia

Projeto de automação com Python para simplificar tarefas diárias e organizar dados de forma eficiente.

# Objetivo do projeto
Este projeto nasceu da necessidade de transformar processos manuais em automações inteligentes. Cada funcionalidade é planejada para facilitar o fluxo do usuário, garantindo eficiência e simplicidade na execução das tarefas.

# Funcionalidades atuais
- Automação de processos com Python
- Organização de dados em planilhas Excel
- Estruturação do código por funções para facilitar manutenção
- Lógica de autenticação e verificação de status do usuário (Playwright)
- Armazenamento de dados local via arquivo JSON

# Tecnologias utilizadas
- Python
- Playwright
- Integração com planilhas Excel
- Automação web e scripts automatizados
- JSON como banco de dados local

# Próximas funcionalidades
- Integração com WhatsApp para envio e recepção de dados
- Funcionalidades de lembretes e alertas para tarefas diárias

## Observações sobre o fluxo do usuário
Cada funcionalidade foi planejada para simplificar tarefas repetitivas, como lembrar compromissos ou organizar informações do dia a dia, garantindo que o usuário tenha experiência intuitiva e eficiente.
